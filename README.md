# Telecom Workforce Dashboard

## Project Overview
This project presents an executive Excel dashboard analyzing over 30 years of global telecom workforce data. The dashboard highlights workforce expansion trends, regional dominance, and the countries contributing most to telecom employment worldwide.

## Dashboard Preview
![Telecom Workforce Dashboard](dashboard.png)

## Key Insights
• Global telecom workforce has grown significantly since the early 2000s.  
• North America leads in telecom employment.  
• The United States is the top contributor to the global telecom workforce.  
• Asia and Europe show strong workforce presence.

## Tools Used
- Microsoft Excel (Pivot Tables, Pivot Charts, Dashboard Design)
- Data Cleaning
- Data Aggregation
- Trend Analysis

## Dataset
Global telecom employment dataset containing country-level workforce statistics across multiple decades.
